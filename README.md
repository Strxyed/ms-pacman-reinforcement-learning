# ms-pacman-reinforcement-learning
Java-based Ms. Pacman AI agents implementing Value Iteration, Policy Iteration, and Q-Learning reinforcement learning algorithms.

## Overview

This project explores AI planning and reinforcement learning by developing agents that learn and optimize gameplay strategies in the Ms. Pacman environment.

Implemented agents:

- Value Iteration Agent
- Policy Iteration Agent
- Q-Learning Agent

## Technologies Used

- Java
- Markov Decision Processes (MDP)
- Reinforcement Learning
- Q-Learning
- Value Iteration
- Policy Iteration

## Features

- State value estimation using Value Iteration
- Policy optimization using Policy Iteration
- Learning agent using Q-learning with exploration
- Performance evaluation against different ghost behaviours
